## Quick Start

> 前提：你的电脑必须安装[Node.js](https://nodejs.org/)才能启动该项目

```bash
awstats init # 初始化 可缩写为: aws i

awstats generate # 生成 aws g (以此类推输入首字母即可)

awstats minify # 压缩

awstats server # 本地预览

awstats deploy # 部署

awstats clean # 清理
```
